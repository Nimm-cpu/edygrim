# Edygrim | Rock DJ Portfolio

A dynamic portfolio website for Nairobi-based Rock DJ, Edygrim. Features a booking system, mix player, and upcoming gigs section.

**Live Demo**: [edygrim.netlify.app](https://edygrim.netlify.app)

## Features
- Fully responsive design
- Interactive audio mix players
- Gig booking form
- Upcoming events display
- Contact integration (WhatsApp, Instagram, Spotify)

## Tech Stack
- HTML5, CSS3, JavaScript
- Custom CSS with rock-themed styling
- Netlify for deployment
- (Adding Supabase as a backend for the booking system soon)
